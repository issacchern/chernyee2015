Research Project "Sapphire"

Ongoing (June 2015 - Present)

Project Sapphire is a research based project under supervision of Professor Alphonce and Professor Ramamurthy. I have developed an eclipse plugin software that captures all the changes under JavaModel in Eclipse application and outputs the data into SQLite database. The purpose of this project is to learn the process of how beginner students start coding and by analyzing the data output style and coding pattern from students, we can improve the quality and effectiveness of instruction at UB.

Check out the project on Github

Bomberman in ARM Assembly

Completed (April 2015 - May 2015)

My partner and I have written and designed the classic Nintendo Bomberman game in ARM assembly language using ARM6 microprocessor and UART serial communication. We use PuTTy to establish the connection with the ARM6 integrated circuit board. We also fully utilize GPIO peripherals such as LEDs, push buttons, 7-Segment display and etc.

Download this project here

Top Down Parsing Tree

Completed (February 2015 - March 2015)

I have developed a toy programming language that followed Chomsky notation with Extended Backus-Naur Form in Java. This program is designed to handle syntatic and semantic errors effectively. It could also be debugging by JIVE - an eclipse plugin extension, which generated a whole object diagram parsing tree from the root object. This project demonstrates the significance of object-oriented programming.

Download this project here
