Project "Sapphire"

Project Sapphire is a research based project under supervision of Professor Alphonce and Professor Ramamurthy. I have developed an eclipse plugin software that captures all the changes under JavaModel in Eclipse application and outputs the data to a file. The purpose of this project is to study the process of how beginner students learn coding. By analyzing the data inputs from students, we can effectively improve the teaching's quality and style.

Check out the project on Github

Bomberman in ARM Assembly

My partner and I wrote and designed the classic Nintendo Bomberman game in ARM assembly language using ARM6 microprocessor and UART serial communication. We used PuTTy to establish the connection with the ARM6 integrated circuit board. We also fully utilized GPIO peripherals such as LEDs, push buttons, 7-Segment display and etc.

Download this project here

Top Down Parsing Tree

I developed a toy programming language that followed Chomsky notation with Extended Backus-Naur Form in Java. This program was designed to handle syntatic and semantic errors effectively. It could also be debugging by JIVE - an eclipse plugin extension, which generated a whole object diagram parsing tree from the root object. This project demonstrated the significance of object-oriented programming.

Download this project here
